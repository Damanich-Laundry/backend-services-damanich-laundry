## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file based on `.env.example`:

example :

```env
PORT = 3000

DB_USERNAME=postgres
DB_PASSWORD=your_password
DB_NAME=database_development
DB_HOST=127.0.0.1
DB_DIALECT=postgres
DB_PORT=5432
```

3. Run the development server:

```bash
npm run dev
```
